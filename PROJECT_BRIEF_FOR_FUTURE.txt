5. HOW DOES IT WORK? (8-STEP PROCESS)
================================================================================

STEP 1: DATA CREATION
- Creates 100 synthetic student records
- Each record has: Study Hours (1-10), Attendance (50-100%), Previous Score (40-90)
- Randomly generated realistic values

STEP 2: MARK CALCULATION
- Formula: Marks = (5 √ó Study Hours) + (0.3 √ó Attendance) + (0.5 √ó Prev Score) + Noise
- Shows that study hours contribute most to final marks
- Adds random noise to simulate real-world unpredictability

STEP 3: DATA SPLITTING
- Divides 100 students into: 80 for training, 20 for testing
- Ensures model is evaluated on unseen data
- Prevents overfitting

STEP 4: MODEL TRAINING
- Algorithm: Linear Regression
- The model learns the relationship between inputs and outputs
- Uses training data to calculate coefficients

STEP 5: PREDICTIONS
- Uses trained model to predict marks for test students
- Compares predicted values with actual values
- Measures prediction accuracy

STEP 6: EVALUATION
- R¬≤ Score: 0.8731 (87.31% accuracy)
- Mean Squared Error: 52.54 (average prediction error)
- Shows model performs EXCELLENTLY

STEP 7: VISUALIZATION
- Creates 2 professional graphs:
  * Left: Scatter plot of Actual vs Predicted marks
  * Right: Heatmap showing feature correlations
- Beautiful styling with light blue background
- Saved as PNG image

STEP 8: CUSTOM PREDICTIONS & EXPORT
- Predicts marks for 3 example students
- Exports results to CSV files for future analysis
- model_results.csv: Model metrics
- predictions.csv: All predictions with errors
